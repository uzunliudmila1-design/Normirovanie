# Правила нормирования

Этот файл читается системой автоматически при каждом анализе.
Правила в разделе «Активные правила» передаются LLM как обязательные ограничения.
Раздел «Случаи для пересмотра» заполняется системой — не редактировать вручную.

---

## Активные правила

- Отверстия в листовой детали (плазма/лазер): если деталь — листовая (тип: листовая, профиль, ребро, щека, фланец, пластина) и изготавливается газо-плазменной или лазерной резкой, то все отверстия вырезаются той же резкой. Операции «Сверлильная» и «Фрезерная» в маршрут не включать — они не нужны.
- Выбор метода резки по толщине листа (СТП 005-01): толщина ≤ 12 мм → «Лазерная резка»; 12 < толщина ≤ 40 мм → «Плазменная резка»; 40 < толщина ≤ 200 мм → «Газовая резка». В диапазоне 10–12 мм альтернативно допустима «Плазменная резка» (выбирать по загрузке оборудования).
- Зачистка после резки (СТП 005-01): операция «Зачистка» обязательна после резки, если толщина листа ≥ 10 мм. Без зачистки маршрут неполный.
- Рихтовка по геометрии (СТП 005-01): операция «Рихтовка» включается в маршрут только если выполнены ОБА условия: 4 ≤ толщина ≤ 40 мм И длина ≥ 300 мм. Иначе «Рихтовка» в маршрут не включается.
- Очистка дробеметная не может стоять до сварки: операция «Очистка дробеметная» (или пескоструйная) всегда выполняется после всех сварочных операций. Размещение очистки до сварки — ошибка маршрута.
- Очистка дробеметная не применяется к деталям: «Очистка дробеметная» используется только для сборочных единиц (изделий), не для отдельных деталей. Для деталей этой операции в маршруте не должно быть.
- Очистка дробеметная не нужна для тонкого металла: если толщина детали менее 10 мм — «Очистка дробеметная» не включается в маршрут.
- Сборочный чертёж (СБ): если чертёж является сборочным, первая операция маршрута — «Комплектовочная», последняя — «Контрольная ГП». Этот порядок обязателен.
- Контрольная ГП только для готовых изделий: «Контрольная ГП» включается в маршрут только если в технических требованиях чертежа указаны покраска и маркировка. Для промежуточных деталей — «Контрольная» (без «ГП»).
- Прихватка перед сваркой: операция «Прихватка» всегда предшествует основным сварочным операциям. Это отдельная операция, не путать с «Сборка/Разборка».
- «Сборка/Разборка» — только для механической сборки без сварки: если в изделии есть сварные швы, используется «Прихватка» + «Сварка полуавтоматическая», а не «Сборка/Разборка».
